# 基于JSP酒店管理系统

#### Description
管理员角色包含以下功能：管理员登录,管理员管理,会员管理,酒店信息管理,酒店预定管理,新闻管理,留言管理等功能。
用户角色包含以下功能：用户首页,新闻公告查看,酒店查看,用户登录注册,留言功能,预定房间,查看订单等功能。
用了技术框架： HTML+CSS+JavaScript+jsp+mysql
